Hinweise zum Praktikum Algorithmen und Datenstrukturen SS 2017 - FH-Aachen

Dieses Repositorium dient dazu die Aufgaben aus dem Praktikum Algorithmen und Datenstrukturen SS 2017 - FH-Aachen zu vestehen,
und bietet die Möglichkeit bei Schwierigkeiten eigene Lösungswege zu Entwickeln.

Es gilt keine Garantie auf die Aufgaben die hier gelöst sind das Praktikum zu bestehen.
Fragen zu den Aufgaben können über den Link https://github.com/BigETI/ADSPrak/issues gestellt werden.
